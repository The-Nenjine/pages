> ord(str, [index], [count])

Convert one or more characters from 
a string to their ordinal character codes.

Returns a string, a table for multiple 
characters, or nil if trying to return a 
value outside of the string.

'str' is a string.

'index' is the character to start from.

'count' is the number of characters to 
convert.
