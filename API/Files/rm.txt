> rm(path)

Deletes a file or folder.

When deleting a folder, is contents 
are recursively  deleted.

If a mount point is deleted the contents 
of the origin folder are not deleted, 
unless explicitly deleted.

See pwd() for more infomation on how 
paths are formatted to be relative or 
absolute.
